git init -> Inicializa un repositorio.
git add #archivo -> Pasa los archivos del workspace al staging area 
                    (se puede usar . en vez del nombre del archivo) de esta manera pasa todo lo que contenga el workspace a la staging area.

git commit -m "mensaje" -> Pasa los archivos del staging area al repositorio.

git status -> Me permite ver el estado actual de mis cambios.
git log -> Me permite ver los commits guardados.
            * --oneline -> me muestra los commits de manera mas simplificada.
            * >archivo.txt -> los exporta a un archivo.
git checkout #hast -> Me permite regresar a un commit anterior.

git branch nombre_rama -> Crea una nueva rama.
git branch -> Me permite ver las ramas existentes y me muestra en cual estoy.
git checkout nombre_rama -> Me posiciona en la rama seleccionada.
